Entraînement sur classes personnalisées

Pipeline prêt pour entraîner YOLOv8 sur un dataset annoté via transfer learning depuis les poids COCO.

Pipeline d'entraînement

  1. Collecter 50–200 images de l'objet cible
  2. Annoter (Roboflow, LabelImg) au format YOLO
  3. Organiser dans dataset/images et dataset/labels
  4. Configurer dataset/dataset.yaml (nc + names)
  5. Lancer python train.py

Paramètres configurés

  • Modèle de base : yolov8n.pt
  • Epochs : 50
  • Image size : 640 px
  • Batch : 16
  • Sortie : runs/train/yolo_custom/weights/best.pt
