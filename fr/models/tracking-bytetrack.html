ByteTrack — Suivi d'objets

Combine la détection YOLOv8 avec un algorithme de tracking pour assigner un ID unique à chaque objet à travers les frames.

Principe

  • model.predict() → détecte sans identité (chaque frame est indépendante)
  • model.track(persist=True) → détecte + maintient l'ID entre les frames

Configuration utilisée

  • Tracker : bytetrack.yaml (intégré Ultralytics)
  • persist : True (état conservé entre les appels)
  • Modèle de détection : yolov8n.pt
  • Script : detect_tracking.py

Exports

  • Vidéo annotée avec IDs : output/tracking_video.mp4
  • Trajectoires CSV : frame, track_id, classe, bbox

Cas d'usage : compter des personnes distinctes dans une vidéo, analyser des trajectoires, vidéosurveillance intelligente.
