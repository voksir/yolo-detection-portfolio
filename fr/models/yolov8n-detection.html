YOLOv8n — Détection d'objets

Modèle nano pré-entraîné sur le dataset COCO (80 classes). Utilisé pour la détection sur image et vidéo.

Caractéristiques

  • Fichier : yolov8n.pt (~6 MB)
  • Architecture : YOLOv8 nano — optimisé vitesse
  • Dataset d'entraînement : COCO (80 classes : person, car, dog…)
  • Sortie : bounding boxes + score de confiance + classe
  • Seuil utilisé : confidence ≥ 0.5

Résultat sur image

Sur input/test_image.jpg (scène de parking) :

  • 4 voitures détectées (classe car)
  • Confiance maximale : 89.5%
  • Confiance minimale : 73.9%

Résultat sur vidéo

Sur input/test_video_1.mp4 → output/test_video_2.mp4 :

  • 721 frames traitées en 1920×1080 @ 30 fps (~24 secondes)
  • Vidéo annotée complète générée par detect_video.py
  • Aperçu disponible sur la page résultats

Scripts associés : detect_image.py, detect_video.py, detect_tracking.py (base detection)

Trade-off vitesse / précision

YOLOv8n est le modèle le plus rapide de la famille. Idéal pour prototypage et CPU. Pour plus de précision, les variantes yolov8s/m/l peuvent être testées en changeant MODEL dans config.py.
