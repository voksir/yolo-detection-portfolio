YOLOv8n-seg — Segmentation d'instance

Au-delà des rectangles, ce modèle produit des masques précis suivant le contour réel de chaque objet.

Différence avec la détection classique

  • Détection : rectangle englobant (bbox)
  • Segmentation : masque binaire + contour polygonal (masks.xy)

Caractéristiques

  • Fichier : yolov8n-seg.pt (~6.7 MB)
  • Sorties : bbox + masques (N, H, W) + polygones
  • Script : detect_segmentation.py

Résultat sur le test

Sur la même image de parking :

  • 4 voitures segmentées
  • Contours de 33 à 234 points selon la complexité de la forme
  • Confiance : 68% à 90%

Cas d'usage : mesure de surface occupée, découpes précises, analyse de forme — là où une simple bbox ne suffit pas.
