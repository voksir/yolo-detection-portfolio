À propos du projet

Ce portfolio présente les résultats visuels d'un projet Python orienté apprentissage de la computer vision. L'objectif : maîtriser le cycle complet — de l'inférence sur données réelles jusqu'à la préparation d'un fine-tuning custom.

🎯

Détection d'objets

Localisation d'objets via bounding boxes sur images et vidéos, avec export JSON/CSV.

Implémenté
🔗

Tracking multi-objets

Suivi persistant des objets entre les frames grâce à ByteTrack et des IDs uniques.

Implémenté
🧩

Segmentation d'instance

Contours pixel-par-pixel des objets avec le modèle YOLOv8-seg.

Implémenté
🧠

Fine-tuning custom

Pipeline d'entraînement prêt pour un dataset annoté (transfer learning depuis COCO).
