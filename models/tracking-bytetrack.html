ByteTrack — Object Tracking

Combines YOLOv8 detection with a tracking algorithm to assign a unique ID to each object across frames.

How it works

  • model.predict() → detects without identity (each frame is independent)
  • model.track(persist=True) → detects + maintains ID across frames

Configuration used

  • Tracker: bytetrack.yaml (built into Ultralytics)
  • persist: True (state preserved between calls)
  • Detection model: yolov8n.pt
  • Script: detect_tracking.py

Exports

  • Annotated video with IDs: output/tracking_video.mp4
  • Trajectory CSV: frame, track_id, class, bbox

Use cases: counting distinct people in a video, trajectory analysis, intelligent video surveillance.
