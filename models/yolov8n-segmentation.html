YOLOv8n-seg — Instance Segmentation

Beyond bounding boxes, this model produces precise masks that follow the actual contour of each object.

Difference from standard detection

  • Detection: enclosing rectangle (bbox)
  • Segmentation: binary mask + polygon contour (masks.xy)

Specifications

  • File: yolov8n-seg.pt (~6.7 MB)
  • Outputs: bbox + masks (N, H, W) + polygons
  • Script: detect_segmentation.py

Test results

On the same parking lot image:

  • 4 cars segmented
  • Contours from 33 to 234 points depending on shape complexity
  • Confidence: 68% to 90%

Use cases: occupied area measurement, precise cropping, shape analysis — where a simple bbox is not enough.
