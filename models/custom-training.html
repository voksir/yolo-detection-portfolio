Training on Custom Classes

Pipeline ready to train YOLOv8 on an annotated dataset via transfer learning from COCO weights.

Training pipeline

  1. Collect 50–200 images of the target object
  2. Annotate (Roboflow, LabelImg) in YOLO format
  3. Organize into dataset/images and dataset/labels
  4. Configure dataset/dataset.yaml (nc + names)
  5. Run python train.py

Configured parameters

  • Base model: yolov8n.pt
  • Epochs: 50
  • Image size: 640 px
  • Batch: 16
  • Output: runs/train/yolo_custom/weights/best.pt
